# store
## 商城项目

### 技术栈
  
  SpringBoot2、mybatis、MySQL、lombok、AspectJ、JQuery。
  
### 实现

  只实现了部分功能：用户注册登录、修改资料、上传头像、收获管理、热销商品、购物车、订单等。
